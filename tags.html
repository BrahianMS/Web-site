Etiquetas HTML

article

Define una sección independiente y autocontenida de contenido, como un artículo de blog, una entrada en un sitio o un comentario.

Ejemplo:

<article> <h2>Título del artículo</h2> <p>Texto del artículo...</p> </article>

nav

Representa una sección de navegación, como menús, tablas de contenido o índices.

Ejemplo:

<nav> <ul> <li> <a href="#">Enlace 1</a> </li> <li> <a href="#">Enlace 2</a> </li> <li><a href="#">Enlace 3</a></li></ul></nav>

header

Define una sección introductoria o de encabezado, que puede contener elementos como títulos, logos o información de autor.

Ejemplo:

<header> <h1>Título de la página</h1> <p>Subtítulo o descripción de la página</p> </header>

ul

Crea una lista desordenada, donde los elementos de la lista (definidos con

  • ) no tienen un orden específico.

    Ejemplo:

    <ul> <li>Elemento 1</li> <li>Elemento 2</li> <li>Elemento 3</li> </ul>

    • table

    Define una tabla, con filas (definidas con ), encabezados (definidos con ) y celdas (definidas con ).

    Ejemplo:

    <table> <thead> <tr> <th>Columna 1</th> <th>Columna 2</th> </tr> </thead> <tbody> <tr> <td>Valor 1</td> <td>Valor 2</td> </tr> <tr> <td>Valor 3</td> <td>Valor 4</td> </tr> </tbody> </table>
